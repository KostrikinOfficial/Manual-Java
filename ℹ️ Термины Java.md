## `☕ Java`
### Объектно-ориентированный язык программирования общего назначения.
___
## `💬 Компилятор javac`
### Программа, которая переводит код, написанный человеком, в форму, пригодную для выполнения компьютером.
___
## `🛠️ JDK`
### Набор инструментов для написания Java-программы.
### Если речь идет о написании и компиляции программы - это JDK.
___
## `▶️ JRE`
### Среда выполнения Java.
### Если речь идет только о запуске программы - это JRE.
___
## `⚙️ JVM`
### Виртуальная машина Java.
### Если говорится о выполнении программы, JIT, Garbade Collector или Class Loader - это JVM.
___
## `📄 Байткод`
### Специальный промежуточный код, который понимает и выполняется JVM.
### Файл с расширением `.class`
___
## `📪 Оперативная память`
### Временное хранилище.
___
## `📦 Переменная`
### Именованная область памяти, предназначенная для хранения значения определенного типа данных.
### Имеет имя, тип и может менять свое значение.
___
## `🧩 Тип данных`
### Атрибут, определяющий какие значения можно хранить,  сколько памяти требуется, какие операции можно выполнять
### Всегда стоит перед именем переменной.
___
## `🏗️ Класс`
### Схема для создания объектов.
### Пишется с заглавной буквы, может содержать поля и методы.
### `String` `Scanner` `User`
___
## `🧱 Поле`
### Переменная, описывающая класс.
### Находится внутри класса, но вне метода.
___
## `⚙️ Метод`
### Блок кода, выполняющий определённое действие.
### Всегда имеет круглые скобки.
### `nextInt()` `println()` `main()`
___
## `👤 Объект`
### Экземпляр класса.
### Создается через `new`
### `Scanner scanner = new Scanner(System.in);` - Объект: `new Scanner(System.in)`
___
## `🔗 Переменная-ссылка`
### Переменная, которая хранит ссылку на объект
### Стоит слева от `new`
### `Scanner scanner = new Scanner(System.in);` - Переменная-ссылка: `scanner`
